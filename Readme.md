
# RoT - Ruby of Thinks


Esempi e codice in Ruby per GPIO e Raspberry Pi.
Installa Ruby e Rails in utente sudo.
Usa rails con utente sudo per far funzionare le gemme.

Esempi applicativi di gemme pi_piper e rpi_gpio
